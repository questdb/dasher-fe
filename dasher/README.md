# DASHER!

The SaaSiest of all.

## Start

Setup:

```bash
npm install
# or
yarn
```

To run development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

Visit [http://localhost:3000/audit](http://localhost:3000/audit) for the task.
